## S-P500_Stock_Performance_Analysis

Using sector data from **yfinance**, I analyzed S&P 500 performance for the first half of 2025. Communication Services and Technology outperformed, while Energy lagged. Results show a cyclical rebound in tech and media versus defensive and energy-focused sectors.
